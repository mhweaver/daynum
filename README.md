# daynum

```
Convert to/from dates and number of days elapsed since 1999-12-13

Usage:
  daynum daynumber
  daynum datestring
where "datestring" is of the form mm([ /-])dd\1yyyy or yyyy([ /-])mm\1dd

When the parameter is a number, daynum will display the date represented by daynumber
When the parameter is a date string, daynum will display the daynumber that represents the date
```
